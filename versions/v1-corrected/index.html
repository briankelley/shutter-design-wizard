Measuring Guide

Inside Mount

  • Height: Measure the window opening height and subtract 1/4"
  • Width: Measure the window opening width, divide by number of panels, then subtract for gaps:
    • 1/16" gap where hinges mount (panel edge to frame)
    • 1/8" gap between adjacent panels

Outside Mount

  • Height: Measure window opening and add desired top/bottom overlap
  • Width: Measure window opening and add desired side overlap, then divide by number of panels
